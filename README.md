# react-stoom-challenge
Plataforma web para realizar pedidos de pizzas

### Dependências
* [Docker](https://www.docker.com/)
* [docker-compose](https://docs.docker.com/compose/install/)
* [NPM e node](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/)
* [npx](https://www.npmjs.com/package/npx)

### Iniciando a aplicação

#### Docker
* Navegar até */app*
* executar `docker-compose build && docker-compose up`
* acessar a rota http://localhost:3001

#### Local
* Navegar até */app*
* executar `yarn install`
* executar `yarn start`
* acessar a rota http://localhost:3000
